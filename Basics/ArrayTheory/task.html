Zoals je kunt zien in de uitwerking van de vorige opdracht wordt er gebruik gemaakt van -1 voor de array. Vergeet niet dat een array bij 0 begint en je dus een cijfermatige opvraag die bij 1 begint zoals in dit voorbeeld (1t/m5) moet terugbrengen naar de index uit de array.

Speel vooral nog even met de gegeven code om meer gewend te raken aan hoe je dit kunt oplossen. Hieronder staan ook de andere soorten loops die ook mogelijk waren geweest. Welke je veelal gebruikt is aan jezelf, wij zullen de for of loop (zie vorige opdracht) het meeste gebruiken.

for (let i = 0; i < dishScores.length; i++) {
   totalScore += dishScores[i];
   console.log(dishScores[i] + ' ' + dishScoreNames[dishScores[i] - 1]);
}

for (let score in dishScores) {
   totalScore += dishScores[score];
   console.log(dishScores[score] + ' ' + dishScoreNames[dishScores[score] - 1]);
}

dishScores.forEach(function(score){
   totalScore += score;
   console.log(score + ' ' + dishScoreNames[score - 1]);
});
