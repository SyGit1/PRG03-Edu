Welkom bij deze op maat gemaakte cursus Javascript binnen de module Programmeren 3. We gaan in alle oefeningen aan de slag om de verschillende aspecten van native Javascript te behandelen. We gebruiken dus geen libraries (jQuery, underscore, lodash, etc.) en ook geen framework (React, Vue, Svelte). We doen alles vanuit de basis zodat je uiteindelijk leert hoe Javascript werkt. Wanneer je uiteindelijk een library of framework gaat gebruiken zul je die veel beter begrijpen, omdat je de basis die onder al die tools gebruik wordt al hebt geleerd.

Laten we beginnen! Maak 2 variabelen aan, een string genaamd dish en een number genaamd dishScore en geef ze een eigen waarde.

Denk goed na over het gebruik van let of const om een variabele aan te maken. var is oud en mag je niet meer gebruiken.
