Nu we weten wat Web API's zijn gaan we kijken naar de eerste toepassing van een Web API, namelijk je geo locatie! Er zijn veel applicaties te bedenken waar het handig is iemands locatie te vragen. Om berekeningen te doen van afstanden, het tonen van een locatie op een kaart of om de juiste standaard taal van de website te activeren voor een bezoeker.

Om toegang te krijgen tot de locatie moeten we de navigator.geolocation code gebruiken. De navigator is het object van jouw browser, en 1 van de mogelijke properties is de geolocation.

  • Koppel de 2 momenteel nog lege global variable aan de 2 HTML elementen. Gebruik hiervoor de kennis van het ophalen van HTML elementen uit de les "DOM > Selecting".
  • Koppel nu een click event aan de button die de functie buttonClickHandler uitvoert.
  • Gebruik nu de navigator.geoLocation om bij het ophalen van de huidige locatie de functie showCurrentLocation aan te roepen. Als je twijfelt over welke method je moet gebruiken, kijk dan eens op de MDN documentatie.
    Zoals je in de uitwerking kunt zien gebruiken we naast getCurrentPosition(), ook de watchPosition() functie, dit om te zorgen dat als iemand gaat lopen de locatie netjes update in de HTML.
  • Als laatste stap plaats je de coordination (latitude, longitude) in de HTML van je target element via de volgende notatie: 51.81400579, 4.6125395
