We hebben nu gewerkt met losse stukken code waar we een aantal functionaliteiten hebben uitgeschreven. Als we praten over 'functionaliteiten' dan is de stap naar daadwerkelijke 'functions' snel gemaakt.

Code organiseren op een zo professionele manier is zeer belangrijk. Bij Programmeren 2 heb je al geleerd meerdere bestanden aan te maken en hebben we hier en daar een aantal functies geschreven. Het maken van deze functions gaan we nu uitgebreider toepassen binnen Javascript.

  • Maak een functie waarmee je 2 scores optelt. De lege functie addScore is alvast voor je neergezet.
  • Maak een functie die het gemiddelde berekend van de nummers in een array. Zorg ervoor dat je om te testen de gegeven array dishScores gebruikt, maar de functie moet elke array kunnen verwerken. De lege functie calculateAverageScore is alvast voor je neergezet.
