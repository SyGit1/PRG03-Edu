In Javascript kun je de typeof check gebruiken om te achterhalen of een bepaalde variable of property is wat jij verwacht (bool/number/string/object). Lees hier meer over 'typeof'.

let name = 'Antwan';
typeof name === 'number'; //false
typeof name === 'string'; //true

function sum(a, b) {
   return a + b;
}
typeof sum === 'object' //false
typeof sum === 'function' //true

Je zult steeds vaker een 'tripple =' statement tegenkomen (===) of de ontkennende variant (!==). Wat dit doet is naast het controleren van de waarde ook controleren of het datatype overeenkomt. Bijvoorbeeld de waarde '1' is een string en geen getal. '1' == 1 zal true zijn omdat de types genegeerd worden terwijl '1' === 1, false zal opleveren omdat het een string en number zijn.
