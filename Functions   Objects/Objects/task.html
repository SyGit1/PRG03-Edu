Objecten zijn (nog meer dan Arrays) enorm krachtig en kunnen op verschillende manieren geschreven worden in Javascript. We bekijken hier het gebruik van een 'object initializer' {}.

Doel van het gebruik van Objecten is nog meer structuur brengen in je code. Een object kan bijvoorbeeld een "Hond" zijn die eigenschappen heeft (ras, leeftijd, naam) en daarnaast functionaliteiten heeft (blaffen, eten).

In object georiÃ«nteerd programmeren noemen we eigenschappen 'properties', en functionaliteiten noemen we 'methods'. Eigenlijk kun je dit zien als variabelen (properties) en functies (methods) maar dan gecategoriseerd binnen een object. Zie hieronder een klein voorbeeld van de Hond:

let dog = {
   name: 'Fido',
   age: 5,
   breed: 'Mixed',
   loves: ['walks', 'fetching balls'],
   bark: function() {
      alert("Woof woof");
   }
}
