Nu we geoefend hebben met Javascript basics, gaan we aan de slag met Javascript in de browser. Zoals in de uitleg video was aangegeven is het belangrijk dat je dit project in je XAMPP folder hebt staan zodat je de HTML-bestanden uit die mappen daar kunt openen om te oefenen. Bijgevoegd zit een index.html bestand, open deze in Google Chrome via je localhost om aan de slag te gaan.

Nu je het juiste bestand open hebt staan kun je ook je browser console openen zodat je als het goed is een lijst aan logs voorbij ziet komen die in de startcode van deze oefening staan. Hou er altijd rekening dat je bij het doen van wijzigingen een hard refresh van je browser doet zodat je zeker weet dat jouw laatste wijzigingen zichtbaar zijn.

Met deze setup zijn we klaar om de DOM (document object model) aan te vliegen. Los de volgende opdrachten op, en probeer in eerste instantie zelf te zoeken naar oplossingen:

  • Maak een variable totalPlaceholder aan en plaats hierin de opgehaalde <div> met het ID 'total' die in de HTML staat.
    Kijk of je gebruik kunt maken van document.getElementById() om het juiste element op te halen uit de HTML.
  • Zorg er nu voor dat de inhoud van de div wordt gevuld met het resultaat uit de optelsom (variable total). Zorg ervoor dat dit getal wordt afgerond op 2 cijfers achter de komma.
    Kijk of je het opgehaalde element kunt vullen met html door de innerHTML property te gebruiken.
